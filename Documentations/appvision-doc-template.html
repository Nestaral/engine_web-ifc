2. Variables Organisation

Variables are defined in the form of a tree structure:

  • {System} Variable representing a set of devices
    • {Device 1} Variable representing a device
      • Status Variable to get the state of the device
      • Comm Variable to get the communication state
      • Command Variable to send commands
      • [SubDevice]X Variable representing a sub-device
        • Status Variable to get the state
        • Cmd Variable to command
    • {Device n} Variable representing another device
      • ...

3. Configuration of the Variables

Video Player

The appPlayer[IntegrationName].dll is a video player control for AppVision that displays video streams from [Integration Name] cameras.

Installation

The player DLL must be copied to each AppVision client $Bin directory.

Live Streaming

Real-time video display with main stream (High) and sub stream (Low) support.

Playback

Recorded video playback with variable speed control and backward playback support.

PTZ Control

Full PTZ control including pan, tilt, zoom, focus, and preset management.

Export

Video export to file with date range selection and progress tracking.

Snapshot

Capture still images from live stream or playback.
